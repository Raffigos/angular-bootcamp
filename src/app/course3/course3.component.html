Overview

MySQL:

  • SQL stands for Structured Query Language used for Relational database management system (RDBMS).
  • SQL-based language for querying and managing databases
  • Basic query syntax includes SELECT, CREATE, WHERE, ORDER BY, UPDATE, DELETE clauses.
  • Provides security features such as user authentication and access control
  • Supports transactions to ensure data integrity and consistency

PHP:

  • PHP stands for Hypertext Preprocessor
  • PHP is a server-side scripting language that is used to create dynamic web pages.
  • PHP can be embedded in HTML pages and executed on the server before the page is sent to the client's web browser.
  • PHP can be used to interact with databases such as MySQL to retrieve and manipulate data.
  • PHP code is executed only when a request is made to the server, making it suitable for web applications.

Common Features:

  • Both languages can be used for back-end communication.
  • Both languages have the function of reading and inserting data from the database.
  • Both languages can be used to secure the sensitive information such as password hashing.

MySQL

Comment is represented by (--).
SELECT * FROM table --Select all fields of the table.
SELECT * FROM table1, table2; --Select more than one table.
SELECT field1, field2 FROM table1, table2; --Select more than one field from different tables
SELECT ... FROM ... WHERE condition --Specify the field(s) using a condition.
SELECT ... FROM ... WHERE condition GROUP BY field --Group the fields that have the same values.
SELECT ... FROM ... WHERE condition ORDER BY field1, field2 --Order the fields by ascending order.
SELECT ... FROM ... WHERE condition ORDER BY field1, field2 DESC --Order the fields by descending order.
SELECT ... FROM ... WHERE condition LIMIT 10; --Choose top 10 fields only.
SELECT DISTINCT field1 FROM ... --Remove the duplicates of fields.
SELECT ... FROM t1 JOIN t2 ON t1.id1 = t2.id2 WHERE condition; --Merge two tables together.
SELECT ... FROM t1 LEFT JOIN t2 ON t1.id1 = t2.id2 WHERE condition --Merge from left table.
SELECT ... FROM t1 LEFT JOIN t2 ON t1.id1 = t2.id2 WHERE condition --Merge from right table.
CREATE DATABASE DatabaseName; --Create a new database.
USE DatabaseName --Select the current database.
DROP DATABASE DatabaseName --Delete the database
CREATE TABLE table (field1 type1, field2 type2, PRIMARY KEY (field1)); --Create a new table by adding fields, specifying the data type, and assigning a primary key which must be unique and not null.
CREATE TABLE table1 (fk_field1 type1, fk_field2 type2, ..., FOREIGN KEY (fk_field1, fk_field2) REFERENCES table2 (t2_fieldA, t2_fieldB)) --Adding a foreign key to refer to the table's primary key and link together two or more tables in a relational database.
-- Main Data Types:
INT --integer; DOUBLE(M,D) --decimal value; VARCHAR(N) --Specify the number of characters of a string; TEXT --Multi-line sentences; DATE (YYYY-MM-DD) --Date format
ALTER TABLE table ADD new_name_field1 type1 --Add a new field.
ALTER TABLE table DROP field1 --Remove a field.
UPDATE table1 SET field1=new_value1 WHERE condition; --Edit the field name.
DROP TABLE table; --Delete the table.
DELETE FROM table1 WHERE condition --Delete the specified tuple (row) from the table.
field1 < value1 --Less than
field1 > value1 --Greater than
field1 = value1 --Equal
field1 <> value1 --Not equal
field1 LIKE 'value _ %' --Search for a specified pattern.
field1 IS NULL --Field is empty.
field1 IS NOT NULL --Field cannot be empty.
condition1 AND condition2 --Both conditions must be true.
condition1 OR condition2 --At least one condition must be true.
